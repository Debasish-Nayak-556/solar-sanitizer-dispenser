  │  (Adjust trim pot until OUT+ = 5.00V)   │
  └─────────────────────────────────────────┘

================================================================
  2. BATTERY VOLTAGE MONITOR
================================================================

  VOLTAGE DIVIDER on Analog Pin A0
  ┌─────────────────────────────────────────┐
  │  Battery+  ──►  R1 (10kΩ)              │
  │                      │                  │
  │                    A0 pin               │
  │                      │                  │
  │                  R2 (4.7kΩ)             │
  │                      │                  │
  │                    GND                  │
  └─────────────────────────────────────────┘

  Formula:
    V_bat = analogRead(A0) × (5.0 / 1023) × ((10 + 4.7) / 4.7)
    V_bat = analogRead(A0) × 0.01532

  ┌─────────────────────────────────────────┐
  │  Relay VCC  ──►  5V rail                │
  │  Relay GND  ──►  GND                    │
  │  Relay IN   ──►  Arduino D3             │
  │                                          │
  │  Relay COM  ──►  Battery+ (or 5V rail)  │

  │                                          │
  │  Route silicone tube:                    │
  │    Sanitizer bottle → Pump inlet         │
  │    Pump outlet → Nozzle tip              │
  └─────────────────────────────────────────┘

================================================================
  7. ARDUINO NANO FULL PIN MAP
================================================================

  Pin   │ Direction │ Connected To
  ──────┼───────────┼──────────────────────────────────
  D2    │ INPUT     │ IR Sensor OUT (active LOW)
  D3    │ OUTPUT    │ Relay IN
  D4    │ OUTPUT    │ Green LED anode (via 330Ω)
  D5    │ OUTPUT    │ Red LED anode (via 330Ω)
  D6    │ OUTPUT    │ Piezo buzzer +
  A0    │ ANALOG IN │ Voltage divider mid-point
  5V    │ POWER OUT │ Sensor VCC, Relay VCC, LEDs
  GND   │ GROUND    │ All component GNDs (common)
  VIN   │ POWER IN  │ MT3608 OUT+ (5V regulated)
